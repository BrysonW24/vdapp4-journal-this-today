# Images Directory

## Required Images

### Background Images
- **journal-background.jpg** - Used for welcome and login pages
  - Recommended size: 1920x1080 or higher
  - Should be a warm, inviting image of someone reading/writing in a journal
  - See reference images provided by user for style inspiration

## Adding Images

1. Add your background image to this directory as `journal-background.jpg`
2. The app will automatically use it for the welcome and login screens
3. You can find free stock photos on:
   - Unsplash (https://unsplash.com) - search for "journal", "writing", "reading"
   - Pexels (https://pexels.com)
   - Pixabay (https://pixabay.com)
